п. и. ли моделирование имя существительное тундровый ненецкий язык для задача морфологический парсинга 1. введение для лингвистический исследование по тундровый ненецкий языку, для уточнение художественный перевод на русский язык быть создать корпус текст на тундровый ненецкий языке. основной источник текст – «фольклор ямальский ненцев» [янгасова, кошкарёв 2018]. корпус состоять из письменный текст эпический фольклорный произведение – сюдбабц, по свой структура являться двуязычным, параллельным, глубина выравнивание являться слово. для создание корпус использоваться программа для анализ язык fieldworks language explorer, разработка летний институт лингвистика (sil international). выбор программа fieldworks language explorer обусловить тем, что программа подходить для анализ и документирование «малоресурсных» (low-resource languages) и малоизученный язык (less-studied languages) [lockwood 2015]. в fieldworks language explorer морфологический парсер xample работать по принцип «единица в окружении» (item-and-arrangement approach): программа справа налево обращаться к поверхностный структура слово и предлагать возможный разметки, то есть, в каждый лексический статья необходимо перечислять всё возможный алломорф и условие они дистрибуции. моделироваться следующий морфологический концепт языка: 1) система категория (часть речи). часть речь выбираться из каталога, основать на gold anthology. к каждый часть речь мочь быть добавить слот аффиксов, слот аффикс мочь быть составить в шаблон аффикс (порядок следование слотов). в fieldworks language explorer существовать встроить список морфосинтаксический свойств, как и часть речи, основать на gold anthology; 2) лексический статьи. формообразующий аффикс мочь быть ввести в слот аффикс определённый часть речи, деривационный аффикс характеризоваться способность изменять часть речь слова; 3) правило сложения; 4) фонемы. фонема делиться на естественный классы, например, гласный и согласные. мочь образовывать вид окружения; входить в фонологический свойства; 5) вид окружения, который мочь быть использовать для распределение алломорфов. для корректный работа морфологический парсер в программа fieldworks language explorer составляться модель тундровый ненецкий язык в категория gold anthology и на материал текст эпический фольклорный произведений. 1. имя существительное тундровый ненецкий
